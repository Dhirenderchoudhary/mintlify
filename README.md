## üöÄ Project Overview

This project is a high-fidelity clone of the Mintlify documentation landing page. It was built to demonstrate proficiency in **modern CSS layout techniques**, **semantic HTML**, and **attention to detail**. The goal was to replicate the design as closely as possible without relying on JavaScript frameworks or utility classes like Tailwind CSS.

**Key Focus Areas:**
*   **Layout Precision:** accurate spacing, alignment, and positioning.
*   **Visual Fidelity:** Matching fonts, colors, and shadows.
*   **Clean Code:** Writing maintainable, well-structured HTML and CSS.
